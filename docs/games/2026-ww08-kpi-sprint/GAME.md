# KPI Sprint

## 한 줄 설명
30초 동안 ‘병목’을 깎아서 KPI 포인트를 최대한 뽑아내는 초단기 운영 게임.

## 룰(2문장)
병목 게이지를 Space/Tap으로 줄여 0을 만들면 점수를 얻고, 곧바로 병목이 다시 생깁니다. 제한 시간 30초가 끝나기 전에 최대 점수를 노리세요.

## 조작(1줄)
- 키보드: Space/Enter
- 모바일: Tap

## Definition of Done (필수)
- [x] 로딩 1초 내(에셋 없음)
- [x] 모바일 터치 동작
- [x] 키보드로 시작/재시작 가능
- [x] 규칙/조작 텍스트로 안내
- [x] 자동 사운드 재생 없음
- [x] LICENSES.md 작성
